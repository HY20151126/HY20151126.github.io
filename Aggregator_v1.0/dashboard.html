计算方式

Current Commission

实时数据，截止查看时间，当月所有此Aggregator关联的设备交易产生的返佣金额之和

单笔返佣 = 交易额 * 返佣比例(有封顶)

金额(当月) = 返佣1+返佣2+...+返佣n

实时数据，截止查看时间，当日此Aggregator所有关联的设备交易产生的返佣金额之和

单笔返佣 = 交易额 * 返佣比例(有封顶)

金额(当月) = 返佣1+返佣2+...+返佣n

Active Agent

实时数据，截止查看时间，当月此Aggregator所有活跃设备绑定的不重复的Agent人数及总绑定设备的不重复的Agent人数

Active Device

实时数据，截止查看时间，当月此Aggregator所有绑定设备中，有激活交易设备的数量

等级

根据上月激活设备数量确定，1级激活数量为100以内，2级激活数量为100以上200以内，3级激活数量为200以上300以内，4级激活数量为300以上，分别对应万7、万8、万9、千1四档返佣比例

根据激活设备数确定

L1：100台以内，含100台

L2:100到200台，含200台

L3:200到300台，含300台

L4:300台以上

交易额

实时数据，截止当前时间，今日的交易金额

非实时数据，昨日交易金额

非实时数据，截止昨天前29天的每日交易金额

交易笔数

实时数据，截止当前时间，今日的交易笔数

非实时数据，昨日的交易笔数

非实时数据，截止昨天的前29天的每日交易笔数

达标率

数据待确认，目前得知的标准为：

Smart POS：日交易额 7W以上

Linux POS：日交易额 5W以上
