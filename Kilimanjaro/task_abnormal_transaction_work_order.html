1. 工单转交功能窗口为工单转交功能通用窗口，即转交类功能均使用此窗口进行操作处理，包括列表页批量操作、详情页操作

3. 列表页批量转交时，当选择的工单全部为单一某种类型时，按照规则2显示；当选择的工单类型不统一时，按照选项少的类型进行展示

工单操作规则说明

1. 工单的编辑、删除仅当工单处于非完结（除Finish外的其他状态）状态时可以进行，仅创建人可以进行。


2. 完结状态的工单不可以进行除查看详情外的所有操作。


3. 导出功能，导出时根据当前筛选的结果进行导出，导出内容为当前筛选条件下的所有结果，结果数上限为5000条。


4. 工单的筛选查询所有拥有该页面权限的用户均可以操作。


5. 批量转交的功能仅创建者和当前处理人可以操作，即当列表选择Mine或To Do时可以使用
